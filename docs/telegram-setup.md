# إعداد إشعارات Telegram للمالك

يوصلك النظام برسائل فورية على Telegram عندما تحدث أحداث مهمة:

- فاتورة مبيعات جديدة
- إغلاق يومي / إغلاق وردية
- خصومات كبيرة، مرتجعات، دفعات من العملاء
- مخزون منخفض
- نتيجة النسخ الاحتياطي
- محاولات دخول فاشلة

---

## الخطوة 1: أنشئ بوتاً عبر @BotFather

1. افتح Telegram على هاتفك أو الكمبيوتر.
2. ابحث عن الحساب الرسمي **@BotFather**.
3. أرسل الأمر:

   ```
   /newbot
   ```

4. اتبع التعليمات لتحديد اسم البوت واسم المستخدم (username).
5. بعد الانتهاء، يعطيك BotFather رسالة تحتوي على **Bot Token** بهذا الشكل:

   ```
   123456789:ABCdefGHIjklMNOpqrSTUvwxyz
   ```

   انسخ هذا التوكن واحفظه في مكان آمن.

---

## الخطوة 2: احصل على Chat ID

1. ابحث عن البوت الذي أنشأته في Telegram.
2. افتح محادثة معه واضغط زر **Start** (أو أرسل أي رسالة مثل "مرحبا").
3. افتح الرابط التالي في المتصفح، مع استبدال `<TOKEN>` بالتوكن الخاص بك:

   ```
   https://api.telegram.org/bot<TOKEN>/getUpdates
   ```

   مثال:

   ```
   https://api.telegram.org/bot123456789:ABCdefGHIjklMNOpqrSTUvwxyz/getUpdates
   ```

4. ستظهر صفحة JSON. ابحث عن قسم `chat` وانسخ قيمة `id`:

   ```json
   {
     "message": {
       "chat": {
         "id": 123456789,
         "type": "private"
       }
     }
   }
   ```

   في المثال السابق، **Chat ID** هو `123456789`.

> ملاحظة: إذا كنت تريد إرسال الإشعارات إلى مجموعة (Group) أو قناة (Channel)، أضف البوت إلى المجموعة/القناة وأرسل رسالة، ثم افتح نفس الرابط وانسخ `chat.id`. قد يبدأ معرّف المجموعة أو القناة بعلامة `-`.

---

## الخطوة 3: اضبط الإعدادات داخل النظام

1. اذهب إلى **مركز الرسائل والحملات** ← **Telegram**.
2. الصق **Bot Token** في الحقل الأول.
3. الصق **Chat ID** في الحقل الثاني.
4. اختر الأحداث التي تريد متابعتها:
   - فاتورة مبيعات جديدة
   - إغلاق يومي / إغلاق وردية
   - الأحداث المهمة
5. فعّل التبديل **تشغيل إشعارات Telegram**.
6. اضغط **حفظ**.
7. اضغط **إرسال تجريبي** للتأكد من أن الرسالة تصل إلى Telegram.

---

## استكشاف الأخطاء

- **فشل إرسال الاختبار**: تأكد من:
  - نسخ Bot Token كاملاً بدون مسافات.
  - نسخ Chat ID بشكل صحيح.
  - أنك بدأت محادثة مع البوت (اضغط Start).
  - وجود اتصال بالإنترنت.

- **الرسائل لا تصل بعد التفعيل**: افتح صفحة `getUpdates` مرة أخرى وتأكد من ظهور `chat.id`. أحياناً يحتاج البوت إلى رسالة جديدة لتحديث `getUpdates`.

- **استخدام بروكسي أو خادم Telegram خاص**: يمكنك تغيير حقل **API Base** إذا كنت تستخدم واجهة برمجة تطبيقات بديلة، أما الافتراضي فيبقى `https://api.telegram.org`.

---

## الأمان

- لا تشارك Bot Token مع أي شخص؛ من يملك التوكن يستطيع إرسال رسائل باسم البوت.
- Chat ID خاص بك ولا يُستخدم إلا لتحديد وجهة الإشعارات.
- يُفضّل تقييد صلاحيات تعديل الإعدادات على حساب المالك أو المدير فقط.
