═══ ElHegazi Retailer — Offline License Activator ═══

A single-file tool for signing licenses on any Windows PC.
No install, no internet, no browser needed.

What you need on your USB:
  1. activator.exe    (this program)
  2. license-private.pem  (your private key)

How to use:
  1. On customer's PC, open the retailer app
  2. The activation screen shows a MACHINE CODE
     e.g.: 7F3A-9C21-BE65-...
  3. Run activator.exe from USB
  4. Paste the machine code when asked
  5. Enter customer/shop name
  6. Choose perpetual or subscription
  7. The ACTIVATION CODE is printed on screen
     → Copy it and paste into the retailer app
  8. The file license.key is also saved in licenses/

Build from source (if you modify the code):
  - Double-click BUILD.bat
  - Or run: npm run build
